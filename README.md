# Курсовая работа. Вариант 3 — Кубики

## Задача
Первый игрок бросает кубики. Сумма на верхних гранях = n.  
Второй игрок получает сумму на нижних гранях.  
Нужно найти **минимум** и **максимум** очков второго игрока.

## Формула
- Противоположные грани в сумме дают 7  
- Если кубиков `k`, то сумма всех граней = `7 * k`  
- Сумма нижних граней = `7 * k - n`

**Минимум:** `k = ceil(n/6)`  
**Максимум:** `k = n`
